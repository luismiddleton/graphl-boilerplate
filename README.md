# Apollo Server 2 (GraphQL) Boilerplate

- MongoDB (using `mongoose`)
- Typescript (`^3.5.3`)
- Babel transpiler

#### MongoDB

Use a local `127.0.0.1` or [Atlas](https://www.mongodb.com/cloud/atlas) instance, which can be configured in `.env`
